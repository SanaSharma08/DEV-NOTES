MULTIPLE DIRECTION STATEMENTS (BOX 5,6)

This works only if you're applying multiple animations on a single element. For example: css Copy Edit animation-name: slideRight, fadeOut; animation-duration: 2s, 3s; animation-direction: normal, reverse; This means: slideRight will play in the normal direction. fadeOut will play in reverse. Each value in animation-direction matches up with the corresponding animation-name by index. If you only have one animation name: Then animation-direction: normal, reverse; will still apply only the first value (normal), and the second (reverse) will be ignored.
