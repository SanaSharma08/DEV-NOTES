THREE WAYS TO CSS IN YOUR WEB PAGE

  1. Inline CSS :

    
                        < p style="color: pink;" >Lorem ipsum dolor sit amet.< /p >

  2. Internal css: 

                            
                            < style > 
                                div{
                                    color: red;
                                    background-color: yellow;
                                }
                            < /style >

  3. External css: use link tag

                            
                            < link rel="stylesheet" type="text/css" href="style.css" >
